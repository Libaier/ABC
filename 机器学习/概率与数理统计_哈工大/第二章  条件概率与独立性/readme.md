# 第7讲 条件概率、乘法定理

* P(A|B) = P(AB)/P(B)
* 乘法定理
	* P(AB) = P(B)P(A| B), P(B) > 0
	* P(AB) = P(A)P(B| A), P(A) > 0
* Polya 模型

# 第10讲 事件的独立性
2.4.1 两个事件的独立性
* 设A，B是两个事件，如果P(AB)= P(A)P(B)，则称A与B相互独立
* 独立与不相容(互斥)不是一个概念
	* A, B相互独立与A, B互不相容不能同时成立.

# 第11讲 重复独立试验、二项概率公式
* 若一个试验只有两个结果:A和A逆, 称试验为伯努利试验.
* 将伯努利试验重复、独立地进行n次, 称为n重伯努利试验.
* 二项概率公式
* 二项概率的泊松(Poisson)逼近定理
