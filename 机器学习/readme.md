##机器学习模型
> 机器学习模型主要从以下几个方面来掌握
> 
> a.产生背景，适用场合（数据规模，特征维度，是否有 Online 算法，离散/连续特征处理等角度）；
> 
> b.原理推导（最大间隔，软间隔，对偶）；
> 
> c.求解方法（随机梯度下降、拟牛顿法等优化算法）；
> 
> d.优缺点，相关改进；
> 
> e.和其他基本方法的对比；
